\doxysubsection{Detailed Description}
Header file for a low-\/level P\+WM library. 

\begin{DoxyDate}{Date}
Aug 25 2020 
\end{DoxyDate}
\begin{DoxyAuthor}{Author}
Amulek1416
\end{DoxyAuthor}
This is a low-\/level P\+WM library that gives a large amount of configuration options that the standard Arduino library doesn\textquotesingle{}t provide.

\begin{DoxyWarning}{Warning}
None of this has been tested due to my current lack of an oscilloscope. However, it does compile. 
\end{DoxyWarning}
